# SmartLight App

Aplicación Android desarrollada como proyecto académico.
Permite el control de una luz LED mediante conexión Bluetooth con un dispositivo Arduino
y la detección de movimiento.

## Funcionalidades
- Encender y apagar luz LED
- Conexión Bluetooth con Arduino (HC-05)
- Detección de movimiento
- Interfaz sencilla y funcional

## Tecnologías utilizadas
- Android Studio
- Kotlin
- Bluetooth
- Arduino

## Requisitos
- Android 7.0 (API 24) o superior
- Dispositivo con Bluetooth
